# ![Focus Lock Logo](icon48.png) Focus Lock

Focus Lock is a simple extension that blocks sites so you can focus on what matters—like studying instead of doomscrolling.

**Focus Lock (free) currently provides:**
- A popup with a simple, clean interface
- Unlimited website blocks at a time
- Automatic redirection when attempting to access a blocked website
- A countdown timer showing the time remaining until the block ends
